Python Data Science Environment Configuration

Posted on 2017-04-03 | |

Anaconda

Preparation

Uninstall the python environment on your platform before install anaconda

Download and install

Anaconda mirror

  • Anaconda2-4.3.1-Windows-x86_64=>python2.7
  • Anaconda3-4.2.0-Windows-x86_64=>python3.5
  • Anaconda3-4.3.1-Windows-x86_64=>python3.6

Be careful

I am using version 3.5 right now,however,Keras only support the version between 2.7 and 3.5 and tensorflow only support the version at 3.5. So it depends on your work. You just need to run the installer and it will install python and other data science tools on your platform automatically.

Test

You have to run a new command line and type python to test the environment.
The command line may output python version if you install anaconda successfully.
sklearn,scipy,numpy,matplotlib and Spyder are included in Anaconda.
